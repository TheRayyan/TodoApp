Prerequisites
1. install nodejs @ https://nodejs.org/en
2. install mongodb @ https://www.mongodb.com/try/download/community
3. install mongodb shell @ https://www.mongodb.com/try/download/shell

How to run this project
1. go to todo-app folder "cd todo-app" and type "npm install"
2. go to backend folder "cd backend" and type "node sever.js"
3. now open separate instance of cmd shell
4. go to frontend folder "cd frontend"
5. type "npm install"
6. after some time type "npm start".

Note : to keep two instance of shell one for backend and another for frontend.
